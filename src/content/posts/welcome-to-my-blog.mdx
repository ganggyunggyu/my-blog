# Next.js 블로그에 오신 것을 환영합니다! 🎉

이 블로그는 **Next.js 15**와 **MDX**를 사용하여 만들어졌습니다. 미니멀한 디자인과 강력한 기능을 갖추고 있어요.

## 주요 기능

이 블로그는 다음과 같은 기능들을 제공합니다:

### 1. MDX 지원

Markdown에 JSX를 더한 **MDX**를 완벽하게 지원합니다. 일반 마크다운 문법은 물론, React 컴포넌트까지 사용할 수 있어요!

### 2. 다크 모드

우측 상단의 버튼으로 다크/라이트 모드를 전환할 수 있습니다. 시스템 설정도 자동으로 감지해요.

### 3. 코드 하이라이팅

코드 블록이 아름답게 표시됩니다:

```typescript
interface BlogPost {
  title: string;
  date: string;
  tags: string[];
  excerpt?: string;
  content: string;
}

const createPost = (post: BlogPost) => {
  console.log('새 포스트: ' + post.title);
  return post;
};
```

### 4. 반응형 디자인

모바일, 태블릿, 데스크톱 모든 화면 크기에서 완벽하게 작동합니다.
